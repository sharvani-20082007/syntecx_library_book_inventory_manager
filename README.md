# Library Book Inventory Manager

This is a simple Python project to manage library books.

## Features
- Add new books
- View all books
- Search for a book
- Remove a book from the inventory

## Technologies Used
- Python

## How to Run
1. Install Python on your system.
2. Download the project file.
3. Open Command Prompt in the project folder.
4. Run the program using:

python library_inventory.py

## Project Description
This program helps manage a library inventory by allowing users to add, search, view, and remove book records.
